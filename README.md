# La Delfina Cabañas - Sitio Web

Sitio web profesional one-page para el alquiler de cabañas en Oliveros, Santa Fe.

## Características

- Diseño moderno y profesional inspirado en sitios de referencia
- Totalmente responsive (mobile-first)
- Optimizado para SEO
- Navegación suave entre secciones
- Formulario de contacto integrado con WhatsApp
- Animaciones sutiles y transiciones suaves

## Estructura

- `index.html` - Estructura HTML del sitio
- `styles.css` - Estilos CSS modernos
- `script.js` - Funcionalidad JavaScript

## Secciones

1. **Hero** - Sección principal con imagen de fondo y call-to-action
2. **Sobre Nosotros** - Descripción del lugar y filosofía
3. **Cabañas** - Detalles de cada tipo de cabaña disponible
4. **Servicios** - Lista de servicios incluidos
5. **Tarifas** - Precios por temporada y tipo de cabaña
6. **Testimonios** - Opiniones de huéspedes
7. **Ubicación** - Mapa de Google Maps integrado
8. **Contacto** - Formulario de reservas y datos de contacto

## Tipos de Cabañas

- **Cabaña de Un Dormitorio** - Hasta 4 personas
- **Cabaña de Dos Dormitorios** - Hasta 5 personas
- **Loft de Dos Dormitorios** - Hasta 5 personas

## Tarifas

### Temporada Regular
- Mínimo 2 noches
- Precios desde $70.000 por noche

### Temporada Alta
- Mínimo 4 noches
- Precios desde $90.000 por noche

## Tecnologías Utilizadas

- HTML5
- CSS3 (Variables CSS, Grid, Flexbox)
- JavaScript Vanilla (ES6+)
- Google Fonts (Playfair Display, Inter)
- Google Maps API

## Personalización

### Colores
Los colores principales se pueden modificar en las variables CSS en `styles.css`:

```css
:root {
    --primary-color: #2c5530;
    --secondary-color: #4a7c59;
    --accent-color: #6b9f7a;
    /* ... */
}
```

### Imágenes
Reemplazar las URLs de imágenes en el HTML con las imágenes reales de las cabañas desde Google Maps.

### Información de Contacto
Actualizar el número de WhatsApp y email en:
- `index.html` (sección de contacto)
- `script.js` (función de envío de formulario)
